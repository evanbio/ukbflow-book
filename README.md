# ukbflow-book

Quarto companion for RAP-native UK Biobank analysis with
[`ukbflow`](https://github.com/evanbio/ukbflow).

The `ukbflow` package website and API reference are generated with pkgdown.
This repository is a separate book for practical workflows, module notes,
design context, examples, and review records.

## What's inside

- RAP setup: authentication, metadata access, phenotype extraction, and job monitoring
- Data preparation: decoding, phenotype derivation, pipeline checks, and cohort management
- Analysis workflows: association models, genetic risk scores, and manuscript-ready outputs
